# docker-task-1-webapp

# ✅ **Task 1: Containerize a Web Application (Real Dev Scenario)**

## 🎯 Objective

Deploy a simple web application using Docker.

## 🧩 Scenario

Your team developed a **Node.js / Python Flask app**, and you need to containerize it so it runs consistently across environments.

## 📌 Requirements

* Create a simple app:

  * Node.js (Express) **OR**
  * Python (Flask)
* Create a `Dockerfile`
* Build Docker image
* Run container on port `3000` or `5000`

## 🔧 Expected Steps

* Write Dockerfile
* Use base image (`node`, `python`)
* Copy application files
* Install dependencies
* Expose port
* Run application

## ✅ Expected Output

* Application should be accessible in browser:

  ```
  http://localhost:3000
  ```

  OR

  ```
  http://localhost:5000
  ```

## 💡 Bonus

* Push image to Docker Hub
